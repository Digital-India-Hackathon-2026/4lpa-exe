# ANTAHDRISHTI

ANTAHDRISHTI is an AI-assisted infrastructure inspection platform designed to help government authorities detect visible structural defects and prioritize maintenance.

## Hackathon Prototype Scope

The original concept supports multiple infrastructure assets. For this hackathon, the working prototype focuses on RCC bridges to achieve greater technical depth.

## Core Flow

Bridge image  
→ AI defect detection  
→ Engineer verification  
→ Historical comparison  
→ Relative deterioration tracking  
→ Maintenance prioritization

## Defects

- Cracks
- Spalling
- Corrosion
- Exposed reinforcement

## USP

ANTAHDRISHTI creates a digital inspection timeline for every bridge. It compares current and previous inspections to identify whether visible damage is increasing over time.
